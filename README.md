# 照片批次加註工具

批次在照片上加註文字(日期、座標、檔名或自訂文字),支援預覽。
以 Python 原始碼發佈,由 MyTools Launcher 動態載入嵌入分頁執行。

## 功能

- **照片匯入**:加入個別檔案,或「加入資料夾」(可連續選取多個資料夾、可含子資料夾)。
- **文字來源三種模式**
  1. 統一文字 —— 每張照片加上相同文字,一鍵輸出。
  2. 對照表 —— 讀 csv / xlsx(需含「照片名稱」與「文字」兩欄),依檔名套用各自文字。
  3. 照片資訊 —— 自動讀 EXIF(檔名、拍攝日期 / 時間、GPS 座標),勾選要加註的欄位。
     - 拍攝日期可選 **西元 / 民國** 年制,並自訂格式範本(如 `YYYY年MM月DD日`)。
- **樣式**:預設白字黑框、黑字白框、黃字黑框、紅字白框、純白 / 純黑字;
  「自訂」可調文字色、描邊色與描邊粗細;字型可選系統字型或自訂 .ttf/.ttc/.otf。
  - **我的樣式**:把整組設定存成具名樣式,下次一鍵套用 / 覆蓋 / 刪除
    (存於 `~/.phototool_styles.json`)。
- **大小**:自動(依圖寬比例,適合不同解析度的批次)或固定像素。
- **位置**:左上 / 右上 / 左下 / 右下,含可調邊距。
- **輸出檔名**:同原始檔名,或自訂前綴 + 流水號(可設起始值與位數)。
- **預覽**:縮圖牆(像檔案總管,點縮圖看大圖)+ 第一 / 上一 / 下一 / 最後一張導覽,
  縮圖與大圖皆即時套用目前文字與樣式。

## 結構

```
photoTool/
├── main_frame.py        # Launcher 入口,提供 create_frame(parent)
├── run.py               # 開發時直接執行
├── requirements.txt     # Pillow, openpyxl(Launcher 安裝時自動 pip)
└── app/
    ├── core.py          # 純邏輯:EXIF / 對照表 / 字型 / 繪製 / 存檔
    └── main.py          # Tkinter 介面(四分頁)
```

## 開發執行

```powershell
pip install -r requirements.txt
python run.py
```

## 注意

- GPS / 拍攝日期需照片本身含 EXIF 才讀得到(多數手機原圖有,經處理 / 截圖通常已被移除)。
- 直拍手機照片會依 EXIF 方向自動轉正,文字才不會貼到錯誤的角。
- 描邊粗細與字級皆相對圖寬縮放,同一份設定可套用不同解析度的照片。
